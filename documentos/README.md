# Documentación del Diagrama C4

Este directorio contiene toda la documentación técnica del sistema de pedidos en línea.

## Archivos principales

- **DIAGRAMA_C4_COMPLETO.md** - Documentación detallada de los 3 niveles C4
- **JUSTIFICACION_ARQUITECTONICA.md** - Justificación de cada decisión arquitectónica
- **EJEMPLOS_CODIGO.md** - Implementación en Python de los patrones de diseño
- **RESUMEN_EJECUTIVO.md** - Resumen para ejecutivos y stakeholders
- **GUIA_GITHUB_PAGES.md** - Instrucciones para publicar en GitHub Pages
- **ESTRUCTURA_FINAL.md** - Verificación de la estructura del proyecto
- **INDICE.md** - Índice completo y navegación por audiencia

## Cómo navegar

- **Ejecutivos/PMs**: Ir a RESUMEN_EJECUTIVO.md
- **Arquitectos**: Ir a DIAGRAMA_C4_COMPLETO.md
- **Desarrolladores**: Ir a EJEMPLOS_CODIGO.md
- **Todos**: Empezar por INDICE.md para una visión completa
